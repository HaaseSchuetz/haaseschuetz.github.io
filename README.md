# Haase Schuetz - Professional Web Presence

This repository contains the source code for my professional GitHub Pages website, showcasing my CV, research projects, publications, and professional information.

## Structure
 HaaseSchuetz.github.io/    
├── index.html              # Main landing page    
├── css/    
│   └── style.css           # Main stylesheet    
├── js/    
│   └── main.js             # JavaScript for interactivity    
├── research/    
│   ├── index.html          # Research overview    
│   └── [research-pages]    # Individual research area pages    
├── projects/    
│   ├── index.html          # Projects overview    
│   └── [project-pages]     # Individual project pages    
├── documents/    
│   └── cv.pdf              # Downloadable CV    
├── publications.html       # Full publications list     
└── README.md               # This file    

## Features

- **Professional CV Section**: Detailed academic and professional background
- **Research Areas**: Overview of current and past research focus
- **Projects Timeline**: Visual timeline of major research projects
- **Publications**: Filterable list of peer-reviewed publications
- **References**: Professional references and contacts
- **Contact Form**: Easy way to get in touch
- **Responsive Design**: Works on all device sizes

## Customization

To customize this template:

1. **Update Personal Information**: Edit the profile section in `index.html`
2. **Add Your CV**: Replace the placeholder content in the CV section
3. **Add Research Areas**: Update the research cards with your actual research focus
4. **Add Projects**: Update the timeline with your real projects
5. **Add Publications**: Update the publications list with your work
6. **Add References**: Update the references section with your professional contacts

## Deployment

This site is automatically deployed via GitHub Pages. Simply push changes to the `master` branch and they will be live at:

https://haaseschuetz.github.io
