Multi-modal Sensor Fusion and Semi-automated Labelling Systems

PhD research on deep learning for multi-modal sensor data in automated driving

My PhD research focused on applied deep learning for multi-modal sensor data, particularly in the context of automated driving. I developed methods for sensor fusion and semi-automated data labelling, bridging machine learning research with scalable, production-relevant systems. My work emphasized efficiency and practicality in handling large, complex datasets.

Research Focus

Multi-modal Sensor Fusion

Development of deep learning architectures that combine data from multiple sensor modalities (LiDAR, camera, radar) to improve perception accuracy and robustness in autonomous driving systems.

Semi-automated Labelling

Implementation of machine learning-based approaches to reduce manual annotation effort while maintaining high-quality ground truth data for training autonomous driving systems.

Production-relevant Systems

Bridging academic research with practical, scalable systems that can be deployed in real-world automotive applications.

Key Contributions

  • Deep Multi-modal Object Detection: Developed methodologies for combining LiDAR and camera data to improve object detection performance in autonomous driving scenarios.
  • Semi-automated Annotation Pipelines: Created systems that leverage weak supervision and active learning to reduce manual labelling costs by up to 70% while maintaining annotation quality.
  • Sensor Fusion Architectures: Designed novel neural network architectures that effectively fuse heterogeneous sensor data at different levels of abstraction.
  • Scalable Data Processing: Built pipelines capable of processing petabytes of sensor data efficiently for training and evaluation.

Methodology

My approach combines theoretical rigor with practical engineering:

  • Empirical Validation: Extensive experimentation on real-world datasets with rigorous evaluation metrics.
  • System Integration: Consideration of deployment constraints, computational efficiency, and real-time performance.
  • Iterative Development: Continuous refinement based on feedback from domain experts and real-world testing.
